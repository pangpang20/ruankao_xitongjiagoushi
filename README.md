# 软考系统架构设计师学习资料库

## 📚 项目简介

本项目是针对**软考系统架构设计师考试**的完整学习资料库，采用CRISPE提示词工程方法论，系统化整理了考试所需的全部知识点。每个主题都包含两份文档：

- **CRISPE提示词文档** (`-crispe.md`): 定义了专家角色、任务要求、背景信息、风格要求和生成步骤
- **完整技术文档** (`-技术文档.md`): 基于CRISPE生成的中英文双语技术文档，包含理论、实践、图表和考试要点

### 📊 文档统计

- **总文档数**: 160+ 份
- **知识领域**: 8 大领域
- **文档格式**: 中英文双语，英文在前，中文在后
- **包含内容**: 理论讲解、架构图、代码示例、案例分析、考试要点

1. 企业信息化与电子商务
2. 政府信息化与电子政务

#### 第四阶段：法规与综合（1周）

### 学习方法建议

**📖 每日学习**:
1. 先阅读CRISPE提示词文档，了解知识框架
2. 深入学习技术文档，理解核心概念
3. 重点记忆标注为**[Exam Focus]**的内容
4. 完成文档中的练习题

**🔄 复习策略**:
1. 利用对比表格加深理解（如病毒vs蠕虫vs木马）
2. 背诵记忆口诀（如"传潜隐破触"）
3. 绘制思维导图梳理知识体系
4. 定期做真题检验学习效果

**💡 答题技巧**:
1. 选择题：排除法 + 关键词识别
2. 简答题：结构化答题（定义→特征→示例）
3. 案例分析题：场景识别 + 框架套用
4. 计算题：公式记忆 + 步骤规范

## 📊 考试题型分布

### 上午选择题（75分）
- **基础知识**（40%）：计算机系统、网络、数据库
- **架构设计**（30%）：软件架构风格、设计模式
- **项目管理**（15%）：需求管理、开发方法
- **法规标准**（15%）：安全法规、信息化标准

### 下午案例分析（75分）

## 💡 记忆技巧总结

### 常用记忆口诀
- **病毒五特征**: 传潜隐破触（传染、潜伏、隐蔽、破坏、触发）
- **事件响应**: 准检遏根恢后（准备、检测、遏制、根除、恢复、事后）
- **设计原则SOLID**: 单开里替依（单一职责、开闭、里氏替换、接口隔离、依赖倒置）
- **架构风格**: 管分层MVC微事（管道过滤器、分层、MVC、微内核、事件驱动）

### 公式速记
- **MTBF** = MTTF + MTTR
- **可用性** = MTBF / (MTBF + MTTR)
- **TMR可靠性** = 3R² - 2R³
- **Amdahl定律**: 加速比 = 1 / [(1-F) + F/S]
